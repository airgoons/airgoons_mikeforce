Setup Instructions
==================

Dependencies
------------
1.  Subscribe to Mike Force in the Arma 3 Workshop
2.  Install Arma 3 Tools via Steam

Configuration
-------------
1.  Make a copy of configuration_example.json and rename to configuration.json
2.  Edit configuration.json
        Arma3Tools_Path - The path to Arma 3 Tools in steamapps/common
        MikeForce_Path - The path to Arma 3 Tools steamapps/workshop
        Customizations_Path - The path to the set of customizations which are applied to an extracted official Mike Force pbo
        Output_Path - The path to where the modifified PBO should be copied after packaging
        Output_Filename - The filename of the output PBO
        Cleanup - Delete the working directory post after packaging the output PBO
        Working_Path - The path to the working directory
